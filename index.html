Machine Learning · Predictive Modelling
Wine Quality Prediction
Explored which physicochemical properties of red wine best predict quality ratings. Compared a pruned regression tree against a Generalized Additive Model (GAM) built through forward selection, evaluated on a held-out test set using RMSE, MAE, and correlation.
Result GAM outperformed regression tree on all metrics - RMSE 0.634 vs 0.685, correlation 0.646 vs 0.569. Alcohol content was the strongest predictor across both models.
